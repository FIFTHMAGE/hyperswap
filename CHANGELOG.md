# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-11-11

### Added
- Initial release of Wallet Wrapped
- Multi-chain support for Ethereum, Polygon, Arbitrum, Optimism, and Base
- WalletConnect integration with Reown AppKit
- Animated story cards with Framer Motion
- Comprehensive wallet analytics powered by Covalent API
- Share and download functionality
- Beautiful gradient UI with TailwindCSS
- Transaction statistics and gas tracking
- Token and NFT portfolio display
- DeFi activity tracking
- Wallet personality ranking system
