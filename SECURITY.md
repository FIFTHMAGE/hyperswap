# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within Wallet Wrapped, please send an email to security@your-domain.com. All security vulnerabilities will be promptly addressed.

Please do not publicly disclose the issue until it has been addressed by our team.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Best Practices

- Never commit your `.env` files
- Keep your API keys secure
- Use environment variables for sensitive data
- Regularly update dependencies
